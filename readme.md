# Arduino Mrugająca Dioda (Pin 13)

Projekt demonstracyjny dla Arduino, który steruje miganiem diody LED podłączonej do pinu 13.

## Wymagania
- MainBox zawierający Arduino Uno lub kompatybilną płytkę UNO
- PlatformIO zainstalowane w VS Code
- Biblioteka Arduino (wbudowana w PlatformIO)

## Instalacja
1. Sklonuj repozytorium:
   ```
   git clone https://github.com/uzytkownik/ArduinoMrugajacaDioda.git
   ```
2. Otwórz projekt w VS Code.
3. Wgraj kod na płytkę za pomocą PlatformIO.

## Użycie
- Po wgraniu kodu dioda LED podłączona do pinu 13 będzie migać w cyklu:
  - 500 ms włączona
  - 1000 ms wyłączona
